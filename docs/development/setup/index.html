Development setup

Clone Premail's GitHub repo:

git clone https://github.com/premail/premail

Run node --version to determine your release of Node and install if necessary.

Node versions

Premail requires at least version 12.x. See Premail setup for more.

In your development directory, run:

npm install

If you don't already have syntax highlighting for MJML, there are plugins for Visual Studio Code, Atom, and Sublime Text 3 and 4.
