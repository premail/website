How to support Premail in getting even better​

If you want to say thank you and/or support active development of Premail:

  • 🌟 Add a GitHub Star to the project.
  • Tweet about Premail (@premaildev on Twitter).
  • Consider sponsoring rootwork, Premail's developer, and have your name or logo included here!
  • Blog about the project. We love reviews, and are eager to hear ways we could make things even better.
